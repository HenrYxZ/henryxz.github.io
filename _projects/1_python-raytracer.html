--- name: Sombra date-range: Jan to April 2020 ---

Sombra is a Raytracer made with Python for my Image Synthesis class. The main purpose of it is to be educational, so I focused on a clean code rather than efficiency. You can see the source code on GitHub here: https://github.com/HenrYxZ/sombra

It first started with a 2D shader and evolved into a 3D Raytracer, you can see the shader here: https://github.com/HenrYxZ/2d-simple-shader
