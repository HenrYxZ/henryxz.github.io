--- name: Raytracing with Python date-range: Jan to April 2020 category: tech ---

Sombra is a Raytracer made in Python. The main purpose of it is to be educational, so I focused on a clean code rather than efficiency.

Some features are: Raytracing, Caustics, Pathtracing, Multi-Threads, Depth of Field, Normal Maps, Textures

I also made series of tutorials for developing a Raytracer in Python from scratch and they are written in Spanish. https://sombra.studio/raytracing-para-todos

Source code here: https://github.com/HenrYxZ/sombra
